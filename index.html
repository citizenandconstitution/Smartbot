Idea Description
Object Detection User Friendly Chat Bot

The project aims to build an image recognition chatbot that can analyze uploaded images and respond to user queries about them. The chatbot will recognize objects in the image, understand the user’s messages, and generate relevant, grammatically correct responses. The solution combines image recognition and NLP to provide accurate and context-aware replies.
